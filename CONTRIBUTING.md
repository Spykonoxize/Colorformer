# Contributing

## Setup

1. Create a Python virtual environment.
2. Install dependencies from `requirements.txt`.

## Guidelines

- Add comments only where logic is not obvious.
- Prefer reusable code in `src/colorformer/` for functions used in production or research.
- Do not commit datasets, checkpoints, or generated outputs.

## Pull Requests

- Use clear commit messages.
- Explain model or training changes in the PR description.
- Include before/after notes for any metric-impacting modifications.
